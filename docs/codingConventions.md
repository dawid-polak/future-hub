# Coding Conventions

## Naming

W calym projekcie stosujemy wylacznie **camelCase**:

- zmienne: `userName`, `totalCount`
- funkcje: `getUserData()`, `calculateTotal()`
- pliki: `userService.ts`, `apiHelper.js`
- foldery: `dataModels/`, `apiRoutes/`

Nie uzywamy snake_case, PascalCase (wyjatki: klasy, komponenty React), ani kebab-case.
