    Adresy IP¶

    Lista adresów IP jest dostępna za pomocą modułu devil vhost.

    Aby uzyskać listę wszystkich dostępnych adresów IP należy skorzystać z polecenia: devil vhost list all. Zawarte w tym wykazie są publiczne adresy IP należące do SMALL.PL oraz te wykupione, prywatne adresy należące do użytkownika. Oba spisy są również dostępne osobno.

    Publiczne adresy IP¶

    Lista publicznych adresów IP zostanie wyświetlona po wykonaniu polecenia: devil vhost list public i zawiera adresy możliwe do wykorzystania przez uruchamiane aplikacje.

    Prywatne adresy IP¶

    Informacja

    Prywatny adres IP można zakupić w panelu płatności

    Lista prywatnych, zakupionych adresów IP zostanie wyświetlona po wykonaniu polecenia: devil vhost list private.
